## Course Overview

This advanced course takes everything you learned in OLI-AAI-101 and puts it into practice at scale. You will contribute directly to **Scwripts** -- the open-source, agent-powered education platform you are learning on right now.

The twist: **AI agents are actively co-building this platform and this course alongside you.** You will study how agents generate course content, write backend services, scaffold frontend components, and review pull requests -- then you will extend those same systems yourself.

By the end of this course, you will understand how agentic systems operate inside real codebases, how to architect agent-augmented development workflows, and how to build features that make the platform smarter for the next cohort of students.

---

## Learning Objectives

Upon successful completion of this course, students will be able to:

1. Navigate and contribute to an open-source full-stack codebase (Django + Next.js) using professional Git workflows.
2. Explain how AI agents generate course content, slides, quizzes, and seed data for production applications.
3. Build agent-powered backend services using Django Ninja, streaming APIs, and the Anthropic SDK.
4. Implement advanced agent patterns in production: context-aware tutoring, adaptive content delivery, and agent-assisted code review.
5. Design and build multi-agent pipelines that generate, validate, and deploy educational content autonomously.
6. Extend a gamification and engagement system with agent-driven personalization.
7. Architect agent-in-the-loop development workflows: from issue triage to PR generation to automated testing.
8. Evaluate, test, and monitor agentic features in a live application with real users.
9. Contribute meaningful open-source pull requests that are reviewed, tested, and merged.

## The Open-Source Model

This course operates like a real open-source project:

- The **Scwripts platform** is the shared codebase. Every student forks it, builds on it, and submits pull requests.
- **AI agents** are first-class contributors. You will see agent-generated commits, agent-written tests, and agent-produced course content throughout the repository.
- **Peer review is mandatory.** Every PR requires at least one human review and one agent-assisted review before merging.
- **Your contributions live beyond the course.** Features you build become part of the platform for future students.

## Academic Integrity

All submitted work must be original. AI agents are your collaborators, not your replacements. Every pull request must include a clear description of what the agent did vs. what you did. Students must be able to explain and defend all code they submit. Misrepresenting agent-generated work as fully original will result in disciplinary action per OLI policy.
